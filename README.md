- Main site: [activistchecklist.org](https://activistchecklist.org)

## Local development

Built with [Starlight](https://starlight.astro.build/) using [Astro](https://astro.build/)

Install dependencies and start the docs site locally:

```bash
yarn install
yarn dev
```
