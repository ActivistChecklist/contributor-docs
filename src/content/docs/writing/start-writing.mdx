Use this page as your writing entry point. If you are new here, follow the sequence below in order before you start a full draft.

## Recommended order

<Steps>

    <LinkButton href="/writing/workflow/" icon="right-arrow">
      Open drafting workflow
    </LinkButton>

1. **Read the contributor style guide** so your draft matches voice, tone, and clarity standards.

    <LinkButton href="/writing/style/" icon="right-arrow">
      Open contributor style guide
    </LinkButton>

1. **Read page structure and formatting** so sections and checklist items are organized correctly.

    <LinkButton href="/writing/structure/" icon="right-arrow">
      Open structure and formatting guide
    </LinkButton>

1. **Choose how you will edit and submit changes.** If you do not want to work directly in code, use the visual editor path.

    <LinkButton href="/writing/visual-editor/" icon="right-arrow">
      Open visual editor guide
    </LinkButton>
</Steps>

## Fast path

If you only have 15 minutes:

1. Read the first section of the [drafting workflow](/writing/workflow/).
2. Skim the [style guide](/writing/style/) principles.
3. Review the checklist item pattern in [structure and formatting](/writing/structure/).
4. Start editing with the [visual editor](/writing/visual-editor/) or directly in MDX.
