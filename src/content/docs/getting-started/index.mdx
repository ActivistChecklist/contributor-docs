This site is the **contributor guide** for [Activist Checklist](https://activistchecklist.org/): a digital security resource built for people in social movements. 

## Start here

<Steps>
1. **Read the code of conduct** so you know what we expect in issues, reviews, and other collaboration.

    <LinkButton href="/getting-started/code-of-conduct/" icon="right-arrow">
      Read the code of conduct
    </LinkButton>

1. **Decide your level of anonymity**. Depending on the threats you face, you may be fine having your true identity attached to this project. If you'd prefer to remain anonymous, read our guide on [protecting your privacy](/getting-started/protecting-your-privacy/) and [create an anonymous GitHub account](/reference/anonymous-github/).

1. **Decide how you are going to contribute**, then open the matching entry point and follow that section of the guide.

    - **[Start writing](/writing/start-writing/)** — Editing existing content or writing new guides.
    - **[Start translating](/translating/start-translating/)** — Editing and approving out automatic translations.
    - **[Start coding](/coding/start-coding/)** — Find issues that need work and open a pull request.

</Steps>

## Communication

* **Content and code discussions** happen on [GitHub issues](https://github.com/ActivistChecklist/ActivistChecklist/issues).
* If you want to report a **bug** or request **new content** on the site: use the  [contact form](https://activistchecklist.org/contact/) or open an [issue on Github](https://github.com/ActivistChecklist/ActivistChecklist/issues)
