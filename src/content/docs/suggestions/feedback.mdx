You don't need to write or code anything to help improve this project. If you spotted something unclear, outdated, or missing, or if you have an idea for a new guide, we'd love to hear from you.

## Option 1: Open a GitHub issue (preferred)

If you have a GitHub account, opening an issue is the best way to share feedback. It keeps suggestions organized, lets us ask follow-up questions, and makes it easy to track what we're working on.

<LinkButton href="https://github.com/ActivistChecklist/ActivistChecklist/issues/new" icon="github" variant="primary">Open an issue on GitHub</LinkButton>

When you open an issue, briefly describe what you noticed or what you'd like to see. You don't need to propose a solution. Just tell us what's wrong or what you'd like to see.

PS - If you want to contribute but prefer not to link your real identity to GitHub, you can [create an anonymous GitHub account](/reference/anonymous-github/) first. This is optional and not necessary for most folks.

## Option 2: Contact form

If you'd rather not use GitHub, you can reach us through the contact form on the main site.

<LinkButton href="https://activistchecklist.org/contact/" icon="external" variant="secondary">Use the contact form</LinkButton>

We check this regularly and will follow up if we have questions.
